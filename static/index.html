ASM Web Service

Minimal HTTP server written in assembly: no libc, only Linux syscalls. x86_64 (NASM) and ARM64 (GAS), one binary per arch. Runs in a scratch Docker image and in Kubernetes.

About

Single process, blocking accept loop. Static HTML and sizes are patched into the binary at build time. Built with nasm / aarch64-linux-gnu-as and ld -static -nostdlib; final image is scratch (no shell, no libs).
